## 2 практическая

**Кратко:**
Работа с XML-разметкой, layout-контейнерами, RecyclerView и отображением списков данных

![](./images/xml_is_easy.png)

Суть задания:

### 1 — Экран со списком

На основе проекта из первой практики необходимо создать **экран со списком элементов**.

Список может быть любым по смыслу:
новости, заметки, задачи, товары, пользователи — не принципиально.
На этом этапе данные могут быть захардкожены прямо в коде.

Главное — сам механизм отображения списка.

Экран должен содержать:

* RecyclerView
* отдельный layout для одного элемента списка
* несколько элементов, чтобы список можно было проскроллить

### 2 — RecyclerView

Необходимо реализовать:

* RecyclerView в XML
* Adapter
* ViewHolder

Каждый элемент списка должен содержать **минимум два UI-элемента**, например:

* заголовок + описание
* текст + кнопка
* иконка + текст

Дизайн может быть простым, но аккуратным.
Это не про UI, это про на понимание, что вообще происходит.

### 3 — Layout’ы и контейнеры

В XML-разметке необходимо использовать:

* минимум один `ConstraintLayout`
* минимум один из контейнеров: `LinearLayout` или `FrameLayout`

Важно понимать, **зачем выбран именно этот контейнер**, а не просто «потому что так получилось».

### 4 — Работа с кликами

Добавьте обработку клика по элементу списка.

При клике:

* либо выводите `Toast`
* либо логируйте событие в Logcat
* либо меняйте внешний вид элемента (например, текст)

Логика может быть минимальной, главное — показать, что клик обрабатывается корректно.

### 5 — Структура проекта

Код не должен лежать в одном файле.

Минимально ожидается:

* отдельный Adapter
* отдельная модель данных (data class)
* Activity, отвечающая только за отображение экрана

MVVM пока **не требуется**, до него мы дойдём позже. Реализуете сейчас - будет проще в будущем.

### Критерии приемки

* Проект использует XML, не Jetpack Compose
* Реализован RecyclerView со списком
* Есть отдельный layout для элемента списка
* Реализован Adapter и ViewHolder
* Присутствует обработка клика по элементу
* Код структурирован и читаем
